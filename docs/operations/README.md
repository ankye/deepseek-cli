# Operations Volume / 操作卷

This volume is for maintainers who validate, archive, publish, or prepare releases.

本卷面向负责校验、归档、发布或准备 release 的维护者。

## Documents / 文档

| Document / 文档 | Purpose / 用途 |
| --- | --- |
| [Validation Gates](validation-gates.md) | Required command gates for normal changes, archive work, and release work. / 普通变更、归档、发布的命令门禁。 |
| [Publishing](publishing.md) | CLI package build and npm publishing rules. / CLI 包构建和 npm 发布规则。 |
| [Acceptance Evidence](acceptance-evidence.md) | How to record release/archive evidence. / 如何记录 release/archive 验收证据。 |

## Maintenance Rule / 维护规则

Do not archive or publish based on local confidence. Archive and release work must have command evidence.

不要基于本地信心归档或发布。归档和发布必须有命令证据。
